Background com propriedade shorthand

Tente aumentar e diminuir o tamanho do seu navegador em todas as direções. Você vai perceber que o fundo vai se adaptar para que ele cubra todo o espaço útil da tela. Obtivemos esse resultado usando o parâmetro background-size com o valor cover (cobrir).

O resultado visual desse exercício é exatamente o mesmo do anterior, mas a configuração das CSS foi mais simplificada, usando shorthands. Dá uma olhada lá no arquivo style.css dessa pasta.
