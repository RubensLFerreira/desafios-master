Sombra em Textos

Vamos entender um pouco como funcionam as sombras em textos. Elas são colocadas basicamente usando a propriedade shorthand text-shadow. A ordem correta para os valores é:

  1. Deslocamento lateral (+ direita e - esquerda)
  2. Deslocamento vertical (+ baixo e - cima)
  3. Espalhamento de cor
  4. Cor da sombra

Sombra dura

text-shadown: 2px 2px 0px black

sombra difusa

text-shadow: 2px 2px 2px black;

Sombra perto

text-shadow: 1px 1px 0px black;

Sombra longa

text-shadow: 4px 4px 0px black;

Sombra para cima

text-shadow: 2px -2px 0px black;

Sombra para esquerda

text-shadow: -2px 2px 0px black;

Sombra para cima e esquerda

text-shadow: -2px -2px 0px black;

Sombra transparente

text-shadow: 2px 2px 0px rgba(0, 0, 0, 0.112)

Sombra colorida

text-shadow: 2px 2px 0px red;

DICAS

  • Normalmente, uma sombra é representada para a direita e para baixo.
  • Para sombras próximas, use um espalhamento menor. Para sombras distantes, aumente o espalhamento.
  • Use sombras pretas, evite sombras coloridas na maioria dos casos.
  • Procure usar sombras com uma certa transparência, isso vai permitir que ela se funda com a cor de fundo.
  • Não use sombras em textos muito longos.
  • Não use sombras para textos com letras muito pequenas.
